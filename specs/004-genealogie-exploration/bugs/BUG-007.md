## Description

Précisions géométriques sur le tracé des liens de l'arbre (prolonge BUG-006).

1. **Lien de couple en 2 segments NON jointifs au symbole.** Le trait reliant un couple est fait de
   **2 segments distincts**, chacun allant **d'un membre vers le symbole d'union (⚭) ou d'ex (⚮)**,
   **sans le toucher** (un petit **espace** subsiste entre chaque segment et le symbole).
2. **Filiation détachée du symbole.** Le trait menant aux **enfants** du couple **part du symbole**
   d'union **sans le toucher non plus** (espace entre le symbole et le départ du trait de filiation).
3. **Co-parents non conjoints reliés quand même.** Si un individu a **N parents non conjoints entre
   eux**, l'arbre DOIT tout de même les **relier par une ligne horizontale** (sans symbole) :
   ```text
   p1 - p2 - p3 - p4 - p5
             |
          enfant
   ```
4. **N parents tous conjoints.** Si les N parents sont **tous conjoints**, un **⚭ figure entre chaque
   paire** consécutive (liens en 2 segments non jointifs, point 1) :
   ```text
   p1 -⚭- p2 -⚭- p3 -⚭- p4 -⚭- p5
               |
            enfant
   ```
5. **Position du trait vertical vers l'enfant (médiane).**
   - **Nombre impair** de parents ⇒ le trait vertical part **sous le parent du milieu** (ex. p3 sur 5).
   - **Nombre pair** ⇒ le trait vertical part **sous le segment/symbole central** (entre les deux
     parents du milieu), c.-à-d. **en dessous du trait liant les parents** (non conjoints) ou **en
     dessous du ⚭ central** (conjoints).
